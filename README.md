# CrowdFlow AI 

## Problem
Large-scale sporting venues often face:
- Heavy crowd congestion
- Long waiting times at gates and food stalls
- Poor navigation for attendees

This leads to frustration and poor event experience.

---

##  Solution
CrowdFlow AI is a smart assistant that helps users:
- Find least crowded entry gates
- Identify shortest queues for food
- Navigate efficiently inside the venue

It uses simulated real-time crowd data and intelligent decision-making.

---

## Features
- Smart gate recommendation
- Food queue optimization
- Google Maps navigation links
- AI-based decision logic
- Crowd simulation system

---

## Tech Stack
- Python
- Google Antigravity
- GitHub
- Google Maps (via links)

---

## How It Works
1. User asks a query (e.g., "best gate")
2. System generates crowd data
3. AI analyzes least crowded options
4. Returns optimized suggestion with navigation link

---

## Assumptions
- Crowd data is simulated
- No real-time sensors used
- Google Maps is used via generated links
